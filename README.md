# Gerenciador de Clientes - CrudCrud API

Esta é uma aplicação web simples desenvolvida para praticar o consumo de APIs REST utilizando a **Fetch API** do JavaScript nativo. O sistema permite realizar as operações básicas de cadastro (POST), leitura (GET) e exclusão (DELETE) de clientes, integrando-se temporariamente com o serviço [CrudCrud](https://crudcrud.com/).

---

## Funcionalidades

- **Cadastrar Cliente (POST):** Envia o nome e e-mail informados no formulário para persistência na API.
- **Listar Clientes (GET):** Busca todos os clientes salvos na API e os exibe dinamicamente em tela ao carregar a página ou após modificações.
- **Excluir Cliente (DELETE):** Remove o cliente selecionado tanto da base de dados da API quanto da interface do usuário.

---

## Tecnologias Utilizadas

- **HTML5:** Estruturação da interface e formulários.
- **CSS3:** Estilização responsiva em formato de cartão baseado em componentes visuais simples.
- **JavaScript (ES6):** Manipulação do DOM, tratamento de eventos e consumo assíncrono da API com `fetch`, `.then()` e `.catch()`.
